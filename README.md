# Shoee

> Shoee is an Ecommerce site. That provides men clothes, watches, and shoes.

## Built With
  - React JS & Hooks
  - React-Router-Dom
  - Redux
  - Redux-Persist
  - React-Icons
  - Formik & Yup
  - Swiper JS
  - SASS

  - #### Knowledge of these technologies
    - React JS & Hooks

### Setup
  * Open your command
  * Type "git clone https://github.com/ahmed-al-farouq/freeeze.git"
  * npm i
  * npm run start

### Run the app
  * npm run start

### Deployment
  * npm run build
